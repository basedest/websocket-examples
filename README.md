# Websocket-examples

This repository contains examples of using the WebSocket API in both Node.js and the browser JS.

## Requirements

* Node.js 8.0.0 or higher
* npm 5.0.0 or higher

## Install

```bash
npm install
```

## Run

1. Get the app running:

```bash
npm start
```

2. Open the browser to http://localhost:3000
